VC LUXURY MASSAGE — Mobile Showcase
====================================

קובץ זה נבנה עבורך בתאריך 2025-08-16.
זוהי תצוגה מונפשת מותאמת-מובייל (9:16) שמציגה 3 תמונות עם מעברי Fade+Zoom בצורה יוקרתית.

איך משתמשים:
1) שימו את קבצי המדיה שלכם בתיקיית assets (כעת היא ריקה):
   - תמונות שקופיות חובה:
     - assets/1.jpg
     - assets/2.jpg
     - assets/3.jpg
     - assets/4.jpg
     מומלץ: 1080×1920 פיקסלים ומעלה, יחס 9:16, פורמט JPG/WEBP.
   - אופציונלי: וידאו פתיחה שמנגן פעם אחת בתחילת המצגת
     - assets/1.mp4
     אם אין קובץ וידאו, המצגת תתחיל אוטומטית אחרי כשתי שניות.

2) פתחו את index.html בדפדפן במובייל או במחשב — זה יוצג כמו סרטון אנכי.

3) רוצים MP4?
   - אפשר להקליט את המסך (Screen Recording) בזמן שהאנימציה רצה,
     או להשתמש בכלי "הקלטת דפדפן"/"html to video" להפקת MP4.

התאמות מהירות:
- משך סבב מלא: בקובץ index.html ערכו את :root --duration (ברירת מחדל: 12s).
- טקסטים: ערכו את הבלוק .headline.
- קישור וואטסאפ: ערכו את ה-URL בכפתור .cta (מושבת כברירת מחדל ב-CSS).

בהצלחה!
